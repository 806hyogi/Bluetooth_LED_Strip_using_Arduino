# 💡 아두이노를 이용한 "스마트 신발장" 프로젝트

## 📌포트폴리오 설명
###### 아두이노 우노를 사용하여 스마트 신발장을 만들었습니다.
###### 블루투스 모듈로 LED 타입을 컨트롤하며 어두운 신발장을 LED strip으로 밝게 분위기를 바꿔줍니다.
###### 또한 비오는 날에는 미리 설정한 패턴으로 나가기전에 우산을 까먹지 않게 패턴을 활용합니다.
